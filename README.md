This repository contains a collection of Artificial Intelligence projects and experiments focused on building modern AI systems using Large Language Models and agent-based architectures.

The projects explore different approaches for LLM orchestration, retrieval systems, and multi-agent collaboration, leveraging modern frameworks and emerging standards.

Technologies & Concepts
	•	RAG (Retrieval-Augmented Generation) pipelines
	•	LangChain for LLM application development
	•	LangGraph for structured agent workflows
	•	AutoGen for multi-agent systems
	•	MCP (Model Context Protocol) integrations
	•	A2A (Agent-to-Agent communication) architectures
	•	Vector databases and knowledge retrieval systems
	•	LLM tool usage and orchestration

Goals
	•	Experiment with modern AI architectures
	•	Build practical LLM-powered applications
	•	Explore agent collaboration patterns
	•	Prototype AI workflows for real-world use cases

This repository acts as an AI experimentation lab, where different frameworks, design patterns, and architectures are tested and implemented.
